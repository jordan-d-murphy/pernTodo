# pernTodo
Basic CRUD app in Postgres Express React Node stack


https://youtu.be/ldYcgPKEZC8?si=kJF6CCN4XMGD2YKd


- - - 


#### to run server: 


cd pernTodo/server


nodemon index


#### to run client: 


cd pernTodo/client


npm start


#### to restart nodemon: 


rs


#### to stop nodemon:


control + c


- - - 


#### postgres: 


cd Library/PostgreSQL/16/bin


psql -U postgres


#### List of databases:


\l


#### Connect to DB:


\c DBNameHere


#### Exit:


quit


#### List all Tables in DB:


\dt

# pernTodo
Basic CRUD app in Postgres Express React Node stack


- - - 


to run: 
cd pernTodo/server


nodemon index


to restart nodemon: 
rs


to stop nodemon:
control + c


- - - 


postgres: 
cd Library/PostgreSQL/16/bin


psql -U postgres


List of databases:
\l


Connect to DB:
\c DBNameHere


Exit:
quit


List all Tables in DB:
\dt
